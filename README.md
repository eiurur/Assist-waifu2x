Assist-waifu2x
======

Chrome Extension to help the use of <a href="http://waifu2x.udp.jp/" target="_blank">waifu2x</a>.

It is an extension to be used in conjunction with <a href="https://github.com/eiurur/Renge" target="_blank">this code</a>.
